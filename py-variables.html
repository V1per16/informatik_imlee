3. Variablen – Speicher für Daten

So speicherst du Informationen in Python

Eine Variable ist wie eine Schachtel mit einem Namen. Du kannst etwas hineinlegen – und später wieder herausnehmen.

So erstellst du eine Variable

Schritt 1: Gib der Schachtel einen Namen (z. B. name)
Schritt 2: Setze = (das bedeutet „lege hinein“)
Schritt 3: Gib den Inhalt an (z. B. "Anna")
# Variable erstellen
name = "Anna"
alter = 25
groesse = 1.65
mag_python = True
name → "Anna"
alter → 25
groesse → 1.65
mag_python → True

So benutzt du Variablen

Jetzt kannst du mit print() den Inhalt der Schachtel ausgeben:

print("Hallo", name)
print("Du bist", alter, "Jahre alt")
Hallo Anna
Du bist 25 Jahre alt
Regeln für Variablennamen:
- Nur Buchstaben, Zahlen, Unterstrich
- Kein Leerzeichen
- Nicht mit Zahl beginnen
- Kleinbuchstaben empfohlen

Variablen ändern

Du kannst den Inhalt der Schachtel jederzeit austauschen:

punkte = 0
print(punkte) # 0
punkte = 10
print(punkte) # 10
Deine Aufgabe:
Erstelle 3 Variablen:
- stadt = deine Stadt
- lieblingsfach = dein Lieblingsfach
- note = deine letzte Note (als Zahl)

Gib dann aus:
Ich wohne in [stadt] und mag [lieblingsfach]. Ich habe eine [note] bekommen.
