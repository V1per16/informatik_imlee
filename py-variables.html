3. Variablen – Speicher für Daten

Wie Schachteln mit Namen, in die du Dinge legst

Eine Variable ist wie eine Schachtel mit einem Aufkleber. Du kannst etwas hineinlegen – und später wieder herausnehmen.

1. So erstellst du eine Variable

Schritt 1: Wähle einen Namen (z. B. name)
Schritt 2: Setze = → „lege hinein“
Schritt 3: Gib den Inhalt an
# Variable erstellen
name = "Anna" # Text → String
alter = 18 # Ganze Zahl → Integer
groesse = 1.65 # Kommazahl → Float
mag_python = True # Wahr/Falsch → Boolean
name → "Anna"
alter → 18
groesse → 1.65
mag_python → True

2. So benutzt du Variablen

Jetzt kannst du den Inhalt der Variablen ausgeben:

print("Hallo", name)
print("Du bist", alter, "Jahre alt")
Hallo Anna
Du bist 18 Jahre alt

3. Variablen ändern

Du kannst den Inhalt der Variablen jederzeit austauschen:

punkte = 0
print(punkte) # 0
punkte = 10
print(punkte) # 10

4. Regeln für Variablennamen

  • Keine Leerzeichen → mein name → Falsch
  • Nicht mit Zahl beginnen → 1name → Falsch
  • Nur Buchstaben, Zahlen, Unterstrich → mein_name_2 → Richtig
  • Kleinbuchstaben empfohlen → alter statt Alter
Merke:
"=" bedeutet „der variablen ... den Inhalt ... hinzufügen“
"==" bedeutet „ist gleich (Vergleichsoperator)“ (kommt in einer späteren Lektion)
Deine Aufgabe:
Erstelle 3 Variablen:
- stadt = deine Stadt
- lieblingsfach = dein Lieblingsfach
- note = deine letzte Note (als Zahl)

Gib dann aus:
Ich wohne in [stadt] und mag [lieblingsfach]. Ich habe eine [note] bekommen.
