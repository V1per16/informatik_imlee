1. Vorbereitungen & Umgebung

So startest du mit Python – Schritt für Schritt

Bevor du programmieren kannst, brauchst du Python und einen Editor. Das ist wie ein Werkzeugkasten fürs Coden.

Schritt 1: Python installieren

Gehe zu: python.org/downloads
Klicke auf den großen gelben Button: "Download Python"
Wichtig (Windows): Aktiviere Add Python to PATH!

Schritt 2: VS Code installieren

Lade es hier herunter: code.visualstudio.com
Installiere es wie jedes andere Programm.

Schritt 3: Dein erstes Programm

Erstelle eine Datei: hello.py

# Das ist dein erstes Python-Programm!
print("Hallo, ich lerne Python!")
Deine Aufgabe:
1. Öffne VS Code
2. Erstelle hello.py
3. Schreibe den Code oben rein
4. Drücke Strg + S
5. Öffne das Terminal und tippe:
python hello.py
Hallo, ich lerne Python!
