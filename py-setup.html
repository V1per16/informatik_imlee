1. Setup & Umgebung

So richtest du Python ein – Schritt für Schritt

Bevor du programmieren kannst, brauchst du Python und einen Editor. Wir zeigen dir die einfachsten Wege.

1. Python installieren

Python ist die Programmiersprache. Du lädst sie von der offiziellen Seite herunter.

Schritt 1: Gehe zu python.org
Schritt 2: Klicke auf „Downloads“ → „Python 3.x“
Schritt 3: Führe die Datei aus → Hake „Add Python to PATH“ an!
Warum „Add to PATH“?
Damit kannst du python in der Eingabeaufforderung (CMD) nutzen.

2. Teste die Installation

Öffne die Eingabeaufforderung (Windows: Win + R → cmd) und tippe:

python --version
Python 3.11.5
Funktioniert nicht? → Starte den PC neu oder prüfe „Add to PATH“.

3. Editor wählen

Du brauchst einen Ort, um Code zu schreiben. Empfohlen:

  • VS Code → Kostenlos, beliebt → Download
  • Thonny → Für Anfänger → Download
  • Replit → Online, kein Download → replit.com

4. Dein erstes Programm

Erstelle eine Datei hallo.py und schreibe:

print("Hallo Welt!")

Führe es aus:

python hallo.py
Hallo Welt!
Deine Aufgabe:
1. Installiere Python
2. Erstelle mein_name.py
3. Gib deinen Namen aus
