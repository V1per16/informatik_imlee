2. Der Print-Befehl

So zeigt Python Text und Zahlen auf dem Bildschirm an

Der print()-Befehl ist wie ein Lautsprecher für deinen Computer. Er sagt ihm: „Zeige das auf dem Bildschirm an!“

1. Was sind Datentypen?

Bevor du etwas ausgibst, musst du wissen, was du ausgibst. Python kennt verschiedene Arten von Daten:

Text (String)
Alles, was in "Anführungszeichen" steht → z. B. "Hallo"
Zahl (Integer)
Ganze Zahlen → z. B. 42, -10
Komma-Zahl (Float)
Zahlen mit Nachkommastellen → z. B. 3.14
Wahr/Falsch (Boolean)
Nur zwei Werte: True oder False

2. Wie gibst du sie aus?

Du schreibst einfach print( und dann den Wert ):

# Text → braucht Anführungszeichen
print("Hallo Welt!")

# Zahl → KEINE Anführungszeichen
print(100)

# Mehrere Dinge auf einer Zeile
print("Ich bin", 16, "Jahre alt")

# Wahr oder Falsch
print(True)
Hallo Welt!
100
Ich bin 16 Jahre alt
True
Wichtig: Text muss in " " stehen. Zahlen nicht!
Text: print("42") → gibt die Zahl als Text Datentyp aus
Zahl: print(42) → gibt die Zahl als Zahlen Datentyp aus

3. Neue Zeile mit \n

Willst du einen Zeilenumbruch? Nutze \n:

print("Erste Zeile\nZweite Zeile")
Erste Zeile
Zweite Zeile

Schreibe eine Zeile Code, die folgendes ausgibt:

Mein Name ist Max und ich bin 14 Jahre alt.
