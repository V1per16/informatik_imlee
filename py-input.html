4. Benutzereingabe

Lass den Computer dich fragen – und speichere die Antwort!

Mit input() kann dein Programm den Benutzer fragen – und die Antwort speichern.

So funktioniert input()

# Frage stellen und Antwort speichern
name = input("Wie heißt du? ")
print("Hallo", name, "!")
Wie heißt du? Max
Hallo Max!

Zahlen eingeben? → Umwandeln!

input() gibt immer Text zurück – auch bei Zahlen!
Das heisst, du musst den Input mit dem Befehl int() von einem String in eine Zahl umwandeln, da man keine Strings zu Zahlen addieren kann.

# Falsch: Gibt Text aus
alter = input("Alter: ")
print(alter + 1) # Fehler!

# Richtig: In Zahl umwandeln
alter = int(input("Alter: "))
print("Nächstes Jahr bist du", alter + 1)
Alter: 16
Nächstes Jahr bist du 17
Merke:
- input("Frage") → Text
- int(input(...)) → Zahl
- float(input(...)) → Kommazahl

Beispiel: Rechner

zahl1 = int(input("Erste Zahl: "))
zahl2 = int(input("Zweite Zahl: "))
print("Summe:", zahl1 + zahl2)
Deine Aufgabe:
Schreibe ein Programm, das fragt:
1. "Wie heißt dein Haustier?"
2. "Wie alt ist es?" (als Zahl)

Gib dann aus:
"[Name] ist [Alter] Jahre alt."
