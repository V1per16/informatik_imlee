4. Benutzereingabe mit input()

Lass den Benutzer antworten – mach dein Programm interaktiv

input() ist wie ein Mikrofon. Es wartet, bis der Benutzer etwas eingibt, und speichert die Antwort.

1. Einfache Eingabe

name = input("Wie heißt du? ")
print("Hallo", name, "!")
Wie heißt du? Max
Hallo Max !
Wichtig: input() gibt immer Text zurück – auch bei Zahlen!

2. Zahlen umwandeln

# Falsch: Text + Zahl = Fehler!
alter = input("Alter: ")
print(alter + 1) # "15" + 1 → "151"

# Richtig: In Zahl umwandeln mit int()
alter = int(input("Alter: "))
print("Nächstes Jahr bist du", alter + 1)
Alter: 15
Nächstes Jahr bist du 16
Umwandlungen:
- int() → ganze Zahl
- float() → Kommazahl
- str() → Text

3. Beispiel: Taschenrechner

zahl1 = int(input("Erste Zahl: "))
zahl2 = int(input("Zweite Zahl: "))
print("Summe:", zahl1 + zahl2)
Deine Aufgabe:
Schreibe ein Programm, das fragt:
1. "Wie heißt dein Haustier?"
2. "Wie alt ist es?" (als Zahl)

Gib dann aus:
"[Name] ist [Alter] Jahre alt."
