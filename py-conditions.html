5. Bedingungen mit if-else

Baue Bedingungen in dein Programm ein

Mit if kannst du sagen: „Wenn das stimmt, mach das!“

1. Vergleichsoperatoren – Die Fragen

Bevor du entscheiden kannst, musst du fragen. Das sind die 6 wichtigsten:

5 == 5 # gleich? → True
5 != 3 # ungleich? → True
10 > 7 # größer? → True
3 < 8 # kleiner? → True
18 >= 18 # größer oder gleich? → True
5 <= 10 # kleiner oder gleich? → True
"==" bedeutet „Ist gleich?“
"=" bedeutet „Speichere in Variable“

2. if-else – Die Entscheidung

alter = 16

if alter >= 18:
    print("Du darfst wählen!")
else:
    print("Noch nicht alt genug.")
Noch nicht alt genug.
Einrückung
Alles was passieren soll, nachdem entschieden wurde, muss eingerückt sein.
Code unter if und else muss 4 Leerzeichen nach rechts.

3. elif – Mehr als Ja/Nein

note = 3

if note == 6:
    print("Sehr gut!")
elif note == 5:
    print("Gut")
elif note == 4:
    print("Okay")
else:
    print("Leider nicht bestanden.")

4. and, or, not - Fragen verketten

  • and → beide müssen wahr sein
  • or → mindestens eines wahr
  • not → kehrt um
note = 3
durchschnitt = 4.5

if note >= 4 and durchschnitt >= 4:
    print("bestanden")
elif note <= 4 or durchschnitt <= 4:
    print("Nicht bestanden")
else:
    print("Klasse wiederholen")
Deine Aufgabe:
Frage nach dem Alter → gib aus:
- „Baby“ (0–1)
- „Kind“ (2–12)
- „Teenager“ (13–17)
- „Erwachsen“ (18+)
