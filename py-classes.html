9. Klassen – Baupläne für Objekte

Organisiere deinen Code Objektorientiert

Eine Klasse ist wie ein Bauplan. Ein Objekt ist das fertige Haus.

1. Was ist __init__?

__init__ ist der Konstruktor. Er wird automatisch aufgerufen, wenn du ein Objekt erstellst.

class Hund:
    def __init__(self, name):
        self.name = name # Speichere Name im Objekt

    def bell(self):
        print("Wuff!")

# Objekt erstellen
mein_hund = Hund("Bello")
print(mein_hund.name)
mein_hund.bell()
Bello
Wuff!
self = das aktuelle Objekt
__init__ = wird beim Erstellen aufgerufen
