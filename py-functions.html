8. Funktionen

Wiederverwendbarer Code in Bausteinen

Eine Funktion ist wie ein Mini-Programm, das du immer wieder aufrufen kannst.

1. Funktion definieren

def begruesse(name):
    print("Hallo", name, "!")

# Aufrufen
begruesse("Max")
begruesse("Anna")

2. Rückgabewert mit return

def addiere(a, b):
    return a + b

ergebnis = addiere(5, 3)
print(ergebnis) # 8
return = „Gib das zurück und beende die Funktion“
Deine Aufgabe:
Schreibe eine Funktion rechteck(laenge, breite), die die Fläche berechnet.
Frage nach Länge und Breite → gib Fläche aus.
